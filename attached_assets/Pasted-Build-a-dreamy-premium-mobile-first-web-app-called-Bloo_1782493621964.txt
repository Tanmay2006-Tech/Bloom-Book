Build a dreamy, premium mobile-first web app called **BloomBook** 🌸
Tagline: "Where little moments bloom forever."

This is a deeply personal scrapbook gift app. It must look like a 
human designer spent months on it — NOT like any AI-generated template.
Every pixel should feel intentional, warm, and handcrafted.

---

🚨 DESIGN PHILOSOPHY — READ THIS FIRST

This app must NOT look like:
- Any shadcn/ui default component
- Any Tailwind template
- Any Material Design or Bootstrap UI
- Generic "pastel app" from dribbble templates
- Anything that looks AI-generated

It MUST look like:
- A premium indie app a designer made as a labor of love
- Somewhere between a physical journal, a Moleskine notebook,
  and a Korean aesthetic stationery shop
- Soft, breathing, alive — like petals in wind
- Every section has its OWN personality and color world
- Typography does heavy lifting — sizes, weights, spacing matter
- White space is used aggressively and intentionally
- Nothing is perfectly symmetrical — organic layouts feel human

VISUAL REFERENCES TO DRAW FROM:
- Bear app (typography-led, warm, minimal)
- Unfold app (editorial layout, lots of white space)
- Day One journal (card depth, warmth)
- Korean stationery aesthetic (soft pastels, illustrated details)
- Notion's warmth but way more expressive and visual

  --text-soft:   #A0827A   ← muted text, captions
  --border:      rgba(242,196,206,0.3) ← ultra-soft borders

NEVER use: pure white (#fff), pure black (#000), 
           cold greys, blue-tinted whites

TYPOGRAPHY — this is what makes it feel premium:

  Font 1: "Playfair Display" — headings, section titles
    - Use italic variant liberally
    - Large sizes (48px+) for hero moments
    - Letter-spacing: tight (-0.02em) for big text
    
  Font 2: "Lato" — all body text
    - Weight 300 for captions, dates, metadata  
    - Weight 400 for body text
    - Weight 700 for labels and emphasis
    
  Font 3: "Caveat" — handwritten accents ONLY
    - Use sparingly: card labels, empty state messages,
      little personality moments
    - Never for functional text

SPACING SYSTEM:
  - Generous padding: cards never feel cramped
  - Section headers: massive top margin (48-64px)
  - Card inner padding: 20-24px minimum
  - Between cards: 16px gap
  - Page horizontal padding: 20px

BORDER RADIUS:
  - Cards: 20px
  - Buttons: 100px (fully pill)
  - Input fields: 14px
  - Chips/badges: 100px
  - Modal bottom sheet: 28px top corners only

SHADOWS — soft, colored (not grey):
  - Card default: 0 2px 20px rgba(242,196,206,0.2)

  - Modal: 0 -4px 40px rgba(45,31,26,0.12)

✨ ANIMATION SYSTEM (Framer Motion)

PHILOSOPHY: Animations should feel like breathing, not performing.
Subtle, physics-based, never distracting.

SPRING CONFIG (use for all interactive elements):
  type: "spring", stiffness: 400, damping: 30

PAGE TRANSITIONS:
  Enter: opacity 0→1, y 12→0, duration 0.3s ease-out
  Exit: opacity 1→0, y -8→0, duration 0.2s

CARD ENTRANCE (staggered):
  Each card: opacity 0→1, y 16→0, scale 0.96→1
  Stagger: 0.06s between cards
  Spring: stiffness 300, damping 28

POLAROID CARDS:
  - On mount: random rotate (-4° to +4°)
  - Hover: rotate→0°, scale 1.04, shadow deepens
  - The random rotation must persist (use useMemo with seeded random)

HEART / FAVORITE TOGGLE:
  - Scale 0.8→1.3→1 on toggle
  - Pink particle burst: 8 dots scatter radially
  - Color fill: empty→filled with spring

FAB BUTTON:
  - Resting: gentle scale breath (1→1.04→1, 3s loop)
  - Tap: rotate 45°, 6 SVG petals burst outward and fade
  - Release: rotate back to 0°

BOTTOM NAV:
  - Active icon: scale spring bounce + indicator pill slides
  - Inactive→active: color transition 0.2s

BOTTOM SHEET MODAL:
  - Enter: y 100%→0% with spring (stiffness 350, damping 35)
  - Drag handle: visible, draggable to dismiss
  - Background: blur + dark overlay fades in

SOMEDAY CONFETTI:
  - 24 particles: random colors from palette
  - Each: random direction, rotation, scale
  - Duration: 1.2s, ease-out

CAPSULE ANIMATIONS:
  - Locked: wax seal pulses with scale 1→1.06→1, 2s loop
  - Unlock moment: crack effect (SVG line draw), seal dissolves,
    envelope top flap rotates open (-30°), message fades in
    with character-by-character typewriter

KITCHEN CARDS:
  - Enter: rotate -4°→0°, y 20→0, like pages settling
  - "Attempted" badge: tiny flame flickers (opacity 0.7→1→0.7 loop)

REVIEW CARDS:
  - Enter: drop from y -8, slight rotation, settle with spring
  - "immaculate" select: star burst sparkle SVG animation

LOADING FLOWER:
  - SVG with 6 petals, each petal strokes in sequentially
  - After petals: center circle scales in
  - Total duration: 1.8s

SPLASH SCREEN:
  - Background: bloom-cream
  - Center: loading flower animation plays fully
  - Then: "BloomBook" in Playfair Display italic fades + scales in
  - Tagline: "Where little moments bloom forever." fades in
  - Total: 2.8s → auto dismiss with fade out

MICRO-INTERACTIONS:
  - Every button: scale 0.96 on press
  - Every card tap: scale 0.98 on press
  - Input focus: border color transitions + slight scale 1.01
  - Toggle switches: spring thumb movement
